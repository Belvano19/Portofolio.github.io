About Me

Halo, saya Belvano Azera Putra Triyanto, seorang mahasiswa dari Universitas Amikom Yogyakarta, Prodi Teknik Komputer dengan konsentrasi IoT (Internet of Things). Saya memiliki minat besar dalam IoT dan pengembangan web. Meskipun saya belum memiliki pengalaman kerja profesional, saya telah mengerjakan beberapa proyek menarik yang membantu saya mengasah keterampilan dan menerapkan pengetahuan dari perkuliahan ke dalam praktik nyata.

Selama menjalankan berbagai proyek, saya telah mempelajari pemrograman Python, HTML, CSS, JavaScript, serta penggunaan alat-alat IoT (Arduino, sensor DHT, dan lain-lain). Saya percaya bahwa setiap proyek adalah kesempatan untuk berkembang, dan saya berusaha menciptakan solusi kreatif serta efektif dalam setiap karya saya.

Portofolio ini saya buat untuk berbagi hasil karya yang telah saya kembangkan sejauh ini, mulai dari membuat website, kalkulator menggunakan bahasa pemrograman Python, sensor kelembapan, real-time clock menggunakan Arduino, hingga pengatur kecerahan pada sebuah LED. Saya bersemangat untuk terus belajar dan tumbuh dalam bidang ini serta terbuka untuk kolaborasi baru yang menantang.
